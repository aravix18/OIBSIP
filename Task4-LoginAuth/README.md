## 📁 Project Overview

A Flask-based authentication system that validates user credentials, checks password strength, and manages user sessions.

## 🚀 Features

* Email validation
* Password strength checker
* Show/Hide password toggle
* User registration
* Session handling

## 🛠️ Technologies Used

* Python
* Flask
* Jinja2
* HTML5
* CSS3
* JavaScript

## ⚙️ How to Run

1. Install Flask:

   ```
   pip install flask
   ```
2. Run the app:

   ```
   python app.py
   ```
3. Open browser:

   ```
   http://127.0.0.1:5000/
   ```

## 📌 Conclusion

This project provided hands-on experience in building a secure full-stack authentication system.
