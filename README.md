## 📁 Project Directory

### 🧮 Task 1: Basic Calculator UI

A responsive arithmetic calculator with real-time input handling and basic mathematical operations. Designed using a structured grid layout and clean UI principles.

**Tech Stack:** HTML5, CSS3, JavaScript

---

### 🎬 Task 2: Movie Tribute Page

A visually structured tribute page dedicated to the Tamil film *Meiyazhagan*, focusing on typography, layout design, and content presentation.

**Tech Stack:** HTML5, CSS3

---

### 🎞️ Task 3: Cinema Watchlist Web App

An interactive movie planner that allows users to manage pending and completed films with dynamic UI updates and DOM manipulation.

**Tech Stack:** HTML5, CSS3, JavaScript

---

### 🔐 Task 4: Login Authentication Portal

A Flask-based authentication system with email validation, password strength checking, user registration, and session handling.

**Tech Stack:** Python, Flask, Jinja2, HTML5, CSS3, JavaScript

## ⚙️ Local Setup (For Task 4 Backend)

To run the authentication system locally:

1. Navigate to the folder:

   ```
   Task4-LoginAuth
   ```

2. Install Flask:

   ```
   pip install flask
   ```

3. Run the application:

   ```
   python app.py
   ```

4. Open your browser:

   ```
   http://127.0.0.1:5000/
   ```
